Exemplos

Funcionalidade: mostrando o behave

  Cenário: rodar um teste simples
     Dado que temos o behave instalado
   Quando implementamos um teste
    Então o behave vai testar isso para nós

Especificações Executáveis

@given('que temos o behave instalado')
def step_impl(context):
    pass

@when('implementamos um teste')
def step_impl(context):
    assert True is not False

@then('o behave vai testar isso para nós')
def step_impl(context):
    assert context.failed is False
